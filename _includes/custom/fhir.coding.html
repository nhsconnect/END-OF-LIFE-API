Important: Where the sending system supports a terminology code:
  1. The condition.code element MUST be populated with:
    • the terminology system (e.g. SNOMED-CT)
    • code
    • display value
  2. the condition.code.text field MUST be populated with the display value from the condition.code.display element e.g. ‘Asthma (disorder)’
  3. the condition.code.text field MAY contain free text to capture more detail associated with the condition.code e.g. ‘Patient uses blue inhaler to relieve symptoms’
